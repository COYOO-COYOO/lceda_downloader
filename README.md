# lceda_downloader 
一键下载立创的器件3D模型和原理图封装、PCB封装，AD格式。

### 📑 介绍
该项目主要用于下载立创器件的3D模型和原理图封装和PCB封装，供AD使用，原理图封装还有点小问题，将就用，不够流畅。。。
